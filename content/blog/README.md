# Blog Content Directory

This directory contains your blog posts in MDX format.

## Structure

Each blog post should be a `.mdx` file with the following naming convention:
- `slug.mdx` (e.g., `examinaite-v2-progress.mdx`)

## Front Matter Template

```mdx
---
title: "Your Blog Post Title"
description: "Brief description for SEO and previews"
date: "2024-01-15"
readTime: "5 min read"
coverImage: "/images/blog/your-post-slug.jpg"
tags: ["tag1", "tag2", "tag3"]
---

Your blog content goes here...
```
