# OG (Open Graph) Images Directory

Create social sharing images for each major page. These appear when your site is shared on social media.

## Image Requirements:
- **Size**: 1200x630px (Facebook/LinkedIn standard)
- **Format**: JPG or PNG
- **File size**: Under 1MB for fast loading

## Required Images:

### Main Pages:
- `og_home.jpg` - Home page ("Who's Watching?")
- `og_about.jpg` - About page
- `og_contact.jpg` - Contact page
- `og_projects.jpg` - Projects page
- `og_blog.jpg` - Blog page

### Persona Pages:
- `og_employers.jpg` - Employers page
- `og_investors.jpg` - Investors page
- `og_fans.jpg` - Fans page
- `og_suitors.jpg` - Suitors page
- `og_curious.jpg` - Curious page (stalkers)
- `og_engineers.jpg` - Engineers page

### Core Pages:
- `og_cv.jpg` - CV page
- `og_awards.jpg` - Awards page
- `og_press.jpg` - Press page

## Design Tips:
- Include your name/brand
- Use consistent color scheme
- Add relevant emoji or icons
- Keep text readable at small sizes
- Consider using your professional headshots

## Usage:
These images will be automatically used in the metadata for social sharing. When someone shares your page on Facebook, Twitter, LinkedIn, etc., this image will appear in the preview.
