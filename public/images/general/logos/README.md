# Logos Directory

This directory contains logos and icons for social media platforms and services.

## Required Logo Files:

### Social Media & Communication:
- `gmail-logo.png` - Gmail logo for email contact
- `linkedin-logo.png` - LinkedIn logo for professional networking
- `instagram-logo.png` - Instagram logo for social contact
- `github-logo.png` - GitHub logo for code repositories
- `calendly-logo.png` - Calendly logo for scheduling

### Professional Services:
- `email-icon.png` - Generic email icon
- `phone-icon.png` - Phone/call icon
- `calendar-icon.png` - Calendar/scheduling icon
- `message-icon.png` - Messaging icon

## Specifications:
- **Format**: PNG with transparent background
- **Size**: 64x64px or 128x128px (square)
- **Style**: Clean, professional, recognizable
- **Background**: Transparent

## Usage:
These logos will be used in contact cards, social links, and professional service displays throughout the website, particularly on the Employers page contact section.
