# Profile Pictures

This directory contains profile pictures for the "Who's Watching?" homepage personas.

## Image Requirements
- **Format**: JPG, PNG, or WebP
- **Size**: Square aspect ratio (1:1) recommended
- **Resolution**: Minimum 300x300px, ideally 600x600px for retina displays
- **File naming**: Use descriptive names matching the persona

## Persona Profile Pictures

Place your profile pictures here with these suggested names:

- `employers.png` - Professional/business look
- `investors.png` - Executive/confident look  
- `fans.png` - Fun/creative personality
- `suitors.png` - Personal/attractive photo
- `curious.png` - Mysterious/investigative vibe
- `engineers.png` - Technical/focused look

## Usage
The homepage will automatically use these images for the Netflix-style profile grid. Images will be displayed in rounded squares with hover effects.

## Fallback
If no image is provided, the system will fall back to emoji avatars as placeholders.
